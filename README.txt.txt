## 📌 PHILOSOPHIE

**ERVI** n'est pas un simple linter. C'est un **médecin pour ton code** qui :

- 🩺 Surveille les **signes vitaux** de ta machine (CPU, RAM, température, disque, réseau)
- 🔬 Diagnostique les **maladies** de ton code (erreurs, perf, sécurité, qualité, complexité)
- 💊 Te donne une **ordonnance** avec des solutions concrètes
- 🔍 **Ouvre VS Code à la ligne exacte** de l'erreur
- 🚀 **Scanne des projets entiers** avec tous les cœurs de ton processeur
- 📦 **Analyse requirements.txt** et détecte les dépendances manquantes
- 📄 **Exporte des rapports PDF** professionnels
- 🔌 **Charge des plugins personnalisés** pour étendre ses capacités

> *"Un codeur en bonne santé, c'est un code en bonne santé."*
